# UnALW1
Tool for removing ad and payment engines from J2ME games

![image](prev.png)

Currently supports:
- ALW1/ALW2
- vServ
- Inneractive
- Hovr
- Freexter
- Greystripe (partially)
- Glomo (not all vendors/games)
- LM (Aircel)
- Gameloft
- Infond
- ASGATech
- Sensible Mobiles
- SMS sending removal
